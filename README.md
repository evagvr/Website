<h1>Bucharest guide Website</h1>
<p>The Bucharest Guide is a website that offers tourists an opportunity to get the best experience of the Romanian Capital by giving multiple examples of places or events to go to, an overall map of the Bucharest zones and a short guide of how the public transport works.</p>
